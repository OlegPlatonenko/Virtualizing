## 6. Lists <a name="LST"></a>

```python
#List examples
my_list = [1, 2, 3, 4, 5]
mix_list = [1, 'a', True]

#Call for list element
my_list[0]
my_list[-2]

#List length
len(my_list)

#Call for range
my_list[0:2]
my_list [1:]
my_list[:3]

#Takes step of 2 for each item
my_list[0::2]

#Set items
my_list[0] = 'a'

#Add to the list
my_list.append(6)

my_list + [8, 9, 10]
my_list += [8, 9, 10]

my_list[1:3] = ['b', 'c']
my_list[3:5] = ['d', 'e', 'f']

#Replace entire section with a nothing
my_list = ['a', 'b', 'c', 'd', 5, 6, 7]
my_list[4:] = []

#Remove items from list
my_list.remove('b')
my_list.pop() #remove from the end
my_list.pop(0)#remove from the beginning
```
