# AWS

## AWS Features

- High Availability
- Fault Toletance
- Scalability
- Elasticity

## AWS Components

- AWS = VPC (Virtual Private Cloud) + S3 (Storage)
- VPC = EC2 (Computer Power) + RDS (Database)

## 1. EC2

- **EC2** - Elastic Cloud Computing

**Common use:**
1. Web Hosting
2. Encode
3. Transcode

## 2. RDS

**Common use:**
1. Customer account info
2. Inventory catalog

## 3. S3

**Common use:**
1. MassStorage
2. Long-Term Storage
